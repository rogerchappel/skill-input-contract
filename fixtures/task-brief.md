# Publish Launch Notes

## Outcome

Draft launch notes for a public repository and prepare a dry-run send plan.

## Inputs

- Repository path
- Release notes draft
- Audience description

## Constraints

- Local-only until approval is granted
- Ask for confirmation before posting or emailing

## Actions

- Inspect README and docs
- Write a launch post draft
- Prepare but do not send an email

## Verification

- Run markdown lint if available
- Include links to generated artifacts
